# OpsZilla — DevOps/Platform Engineering MCP Server

**OpsZilla** é um agente especialista em DevOps, Cloud Engineering, Platform Engineering e Observabilidade, responsável por transformar aplicações em ambientes seguros, escaláveis, resilientes, monitoráveis e prontos para produção.

## Filosofia

```
PixelFera desenha a experiência.
FrontZilla constrói a interface.
BackZilla sustenta a lógica.
OpsZilla coloca tudo de pé e mantém saudável.
```

## Conhecimentos

### DevOps
- CI/CD (GitHub Actions, GitLab CI)
- Docker, Docker Compose
- Kubernetes, Helm
- Terraform, Pulumi
- Blue/Green, Canary, Rollback

### Cloud
- AWS (ECS, Lambda, API Gateway, SQS)
- GCP (Cloud Run, GKE, Pub/Sub)
- Azure

### Observabilidade
- Logs estruturados
- Métricas (Prometheus, Grafana)
- Tracing (OpenTelemetry)
- Alertas e health checks
- SLO/SLA/SLI

### Segurança
- IAM, RBAC
- Secrets management
- Scanning de vulnerabilidades
- Least privilege
- Network policies

## Instalação

```bash
cd opszilla-mcp-server
npm install
npm run build
```

## Teste

```bash
npm test
```

## Execução

```bash
npm start
```

## Uso

Acesse via `/mcp` no Claude Code e selecione **opszilla-mcp-server** para acessar todas as 19 tools.
